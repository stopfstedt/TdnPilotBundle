AbstractManipulator

abstract class AbstractManipulator implements ManipulatorInterface

Abstract Class AbstractManipulator

Parent manipulator class. Each manipulator deals with a specific part of a target application (controllers, managers, handlers, routing, etc) When adding a new type of manipulation, a new manipulator should be created extending this class.
