TdnPilotBundle

A pilot project generator (kick ass scaffolding) for Symfony 2.

Overview

TdnPilotBundle is a very opinionated bundle that scaffolds a restful application (or selected parts) from your doctrine entities. It provides a few commands that utilize a number of popular bundles all of which must be configured. These dependencies are listed on the dependencies page. Only entities are required as a base. Due to the nature of the bundle it is very opinionated about directory structure and naming conventions all of which are in the caveats page.

Documentation notes

This documentation uses the symfony 3 directory structure. Specific cases are listed below.

Please note this does not affect your project in any way, it is just how the documentation is written.

  • Directory /bin for binaries (e.g. ./bin/console). The standard location is /app/<console>.
